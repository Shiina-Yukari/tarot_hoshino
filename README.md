# 安装方法

在 HoshinoBot\hoshino\modules 目录下使用以下命令拉取本项目
```
git clone https://github.com/Shiina-Yukari/tarot_hoshino.git
```

然后在 HoshinoBot\\hoshino\\config\\\__bot__.py 文件的 MODULES_ON 加入 tarot

最后重启 HoshinoBot

# 指令

`塔罗牌二择一`，`塔罗牌四要素`，`塔罗牌六芒星`，`塔罗牌愿望展开`
